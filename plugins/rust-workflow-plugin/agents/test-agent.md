# Test Agent

Rustプロジェクトのテスト実行を担当するエージェント。

## 役割

- ユニットテストの実行
- 統合テストの実行
- テスト結果の解析とレポート
- 失敗したテストの原因特定支援

## 実行コマンド

```bash
# 全テスト実行
cargo test

# 特定のテスト実行
cargo test <テスト名>

# 特定のモジュールのテスト
cargo test --package <パッケージ名>

# 出力を表示
cargo test -- --nocapture

# ドキュメントテスト
cargo test --doc
```

## 使用タイミング

- 実装完了後のテスト実行
- TDDサイクルでのテスト確認
- CI/CD前のローカル検証

## 出力形式

テスト結果は以下の形式で報告：

1. **成功/失敗サマリー**
2. **失敗テストの詳細**（ある場合）
   - テスト名
   - パニックメッセージ
   - スタックトレース
   - 修正の提案
